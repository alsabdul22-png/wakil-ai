Welcome to the official repository of the **Elvrona Group**, the strategic parent company overseeing the development of the **Wakil AI Ecosystem**. This project represents a unified, proprietary neural architecture designed for high-performance localized intelligence and autonomous systems.

## 🚀 The Ecosystem at a Glance

The Elvrona Group manages several specialized platforms, each integrated into the core neural framework:

### 1. [Wakil AI - Official](./Wakil/Wakil-Official)
The flagship AI interface. A professional, high-fidelity platform providing direct access to our localized neural models.
- **Tech:** React, Framer Motion, Tailwind CSS.

### 2. [Wakil Shop - Marketplace](./Wakil/Wakil-Shop)
A multi-vendor marketplace integrated with the Wakil ecosystem.
- **Tech:** Multi-language support (10+ languages), standardized branding.

### 3. [Wakil AI Tech - Infrastructure](./wakil-ai-tech)
The backbone of our technology. This project focuses on the low-level infrastructure and technical specifications.
- **Tech:** Next.js (App Router), TypeScript.

### 4. [Wakil AI Chat (NextGen)](./wakil-ai)
Our next-generation chat terminal with "Visual Core Synthesis" (AI Image Generation).
- **Tech:** Next.js, API Integrations (Hugging Face / AI Horde).

### 5. [Robot App - Autonomous View](./robot-app)
Our latest expansion into hardware/software integration.
- **[Mobile View](./robot-app/mobile/index.html)**: Control interface for field operatives.
- **[Restaurant View](./robot-app/restaurant-view/index.html)**: Real-time dashboard for localized robotic service automation.

## 🛠️ Global Development Instructions

### Environment Setup
Each sub-project contains its own `.env` file for API keys (Hugging Face, etc.). Ensure these are configured.

### Running the Ecosystem
- **Main Portal:** Open `index.html` in the root directory.
- **React Apps:** Navigate to the folder and run `npm start` or `npm run dev`.
- **Wakil AI (NextJS):** `npm run dev` in the `wakil-ai` folder.

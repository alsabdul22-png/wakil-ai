# 🌍 Wakil AI Ecosystem

Welcome to the **Wakil AI Ecosystem**, a professional suite of AI-driven applications designed for modern web interaction and digital marketplaces.

## 📁 Repository Structure

This workspace contains three main projects that work together to provide a complete AI ecosystem:

### 1. [Wakil AI Web Application](./wakil-ai)
The core intelligence engine. A **Next.js** application deployed to **Google Cloud Run**.
- **Features**: Interactive AI chat, serverless architecture, responsive UI.
- **Stack**: Next.js 15+, TailwindCSS, SQLite (LibSQL), Google Cloud Run.

### 2. [Wakil Shop](./Wakil/Wakil-Shop)
A modern marketplace for digital services.
- **Features**: Multi-language support (10+ languages), product catalog, shopping experience.
- **Stack**: Vite, React, TypeScript.

### 3. [Wakil-Barada Portal](./wakil%20baeada%20ai%20webseit)
A static landing page for branding and ecosystem overview.
- **Features**: High-performance static content, emerald green glassmorphism design.
- **Stack**: HTML5, CSS3, Vanilla JS.

## 🚀 Getting Started

### Prerequisites
- **Node.js 20+**
- **Google Cloud CLI** (for deployment)
- **npm** (Package manager)

### Development
To run any specific application, navigate to its folder and run:
```bash
npm install
npm run dev
```

## 🏗️ Deployment
The ecosystem is optimized for **Google Cloud**:
- **Core App**: Deployed via Cloud Run (`gcloud run deploy`).
- **Portal**: Hostable on any CDN or Static Host.

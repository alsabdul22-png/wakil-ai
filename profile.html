01 — About

Background

I am an independent developer based in Germany. My focus is on building AI-powered software that is technically honest — systems where the architecture is documented, the limitations are disclosed, and the data handling is transparent.

Wakil AI started as a personal project to explore what a privacy-respectful AI assistant could look like in practice. I built the frontend, the API layer, the authentication system, and the inference routing from scratch. It is a real product, not a demo.

I prefer to build in public and be honest about what works and what does not. If you have questions about how something is built, there is a technical architecture page for that.

Core focus areas

  • Full-stack web development
  • AI systems & prompt engineering
  • Privacy-focused infrastructure
  • Backend API design
  • EU-hosted product deployment

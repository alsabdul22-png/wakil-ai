## 3. USER FLOW: THE JOURNEY TO EXECUTION
1.  **Launch:** User opens the app to the minimal "Neural Point" entry.
2.  **Verify:** Secure login via OTP email verification (ensures account integrity).
3.  **Mode Select:** User clicks one of the four core mode buttons (Reasoning/Synthesis/Research/Code).
4.  **Prompt:** User enters the prompt into the focused, central terminal area.
5.  **Execute:** AI processes; results appear in clean, verified message bubbles.
6.  **Action:** User copies code, saves image, or continues the deep-dive thread.
7.  **Preserve:** Sidebar automatically organizes the session into history for future access.
